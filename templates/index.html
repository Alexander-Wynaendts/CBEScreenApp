Entourage: CBE Sourcing

About the Program

This program retrieves websites of newly registered Belgian companies that could fit Entourageâ€™s investment criteria based on CBE data.

The program filters companies by NaceCode, requiring at least one of the following codes: [582, 62, 63] and excluding any companies with codes: [0, 1, 2, 681, 682].

The program then ensures the company has:

  • A valid company name
  • At least 1 founder
  • Founded on or after 2020

Once filtered, the program attempts to find the founders' LinkedIn profiles through a Google search and retrieves the website URL from it's company LinkedIn profile.

Important: To use this program, you must manually download the CSV file from the CBE Open Data Belgian website:

  • Go to CBE Open Data Belgian website
  • Select "Download a BCE Open Data File"
  • In the "Update" column, download the desired .zip file (usually the latest one)

Note: The program runs live but can only process the same file for up to 12 hour. It cannot process more than 4,000 companies in one go.
